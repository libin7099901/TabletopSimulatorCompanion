# 🛠️ 桌游伴侣 v4.0 快速修复指南

## 🚨 问题分析

报错信息：`Error in Script (Global) function <onLoad> chunk 4:(793,23-25): attempt to call a nil value`

### 主要问题原因

1. **API兼容性问题**：使用了不存在或错误命名的TTS API
2. **错误处理不足**：缺少对API调用的安全检查
3. **事件函数问题**：某些事件监听函数在TTS中不存在

### 具体问题点

1. `Hands.getHands()` - 此API可能不存在或命名错误
2. `onObjectEnterContainer` - 此事件函数不存在
3. 缺少对各种API调用的错误处理
4. 某些Player API调用未进行安全检查

## ✅ 解决方案

### 1. 使用修复版脚本

已创建 `TabletopCompanion_v4_Fixed.ttslua`，修复了所有已知问题：

**主要修复内容：**
- ✅ 移除了有问题的 `Hands.getHands()` API调用
- ✅ 简化了对象信息收集，使用安全的API
- ✅ 移除了不存在的事件监听函数
- ✅ 添加了全面的错误处理和pcall保护
